How to convert HttpException to RpcException?

It is really easy to return erroneous http response in Nest.js application, you just have to throwHttpException or one of it`s implementations like UnauthorizedException,NotFoundException or BadRequestException. It is meaningful and you don`t have to reinvent the wheel each time. But What about microservices? Potentially you can throw the same Exception from there too, but on the other side you will receiveInternal Server Error, which is not meaningful at all, so to use built-in exceptions they has to be converted to RpcException first. To do this I wrote this simple filter.

import {Observable, throwError} from "rxjs";
import {Catch, RpcExceptionFilter, HttpException} from "@nestjs/common";
import {RpcException} from "@nestjs/microservices";


@Catch(HttpException)
export class HttpToRpcExceptionConverter implements RpcExceptionFilter<HttpException> {
  catch(exception: HttpException): Observable<any> {
    return throwError(new RpcException(exception.message));
  }
}
